Vangrapf

Documentation — watch and download YouTube videos via proxy

Vangrapf is a web interface that lets you search, watch, and download YouTube videos through a proxy server. It helps bypass regional blocks and removes ads. You can use either a public proxy list or your own server.

Quick Start

  1. Open the Vangrapf page in your browser.
  2. Set up a proxy (optional) – click the ☰ icon in the top-right, enter your proxy URL (e.g., https://your-proxy.onrender.com), and save. If left empty, the site will automatically pick a working proxy from the public list.
  3. Search for a video – type a query in the search bar and press Enter or click Search.
  4. Choose an action – for each video you can:
    • Copy link – click the video card or the “Copy link” button. This copies a stream URL like https://proxy/stream?url=https://youtube.com/watch?v=....
    • Download – click the “Download” button. A progress window appears and the video will be saved as video.mp4.

Using the Copied Stream URL with Media Players

MPV (recommended)

MPV is a fast, lightweight player with excellent support for streaming and seeking (Range requests).

  1. Install MPV for your OS.
  2. Open a terminal and run: 
    mpv "https://your-proxy/stream?url=https://www.youtube.com/watch?v=dQw4w9WgXcQ"
    Replace the URL with the one you copied.
  3. Control playback with standard MPV keys (Space to pause, arrow keys to seek, etc.).

VLC media player

VLC can open network streams directly.

  1. Launch VLC.
  2. Go to Media → Open Network Stream (or press Ctrl+N).
  3. Paste the copied stream URL into the network address field.
  4. Click Play. The video will start streaming.

Windows Media Player (WMP)

WMP does not support network streaming directly. However, you can:

  • Use the Download button on the Vangrapf site to save the video as an MP4 file, then open it in WMP.
  • Alternatively, install a more capable player like MPV or VLC.

Xplayer (Android)

If you copy Vangrapf video link, Xplayer found it.

  1. Launch Xplayer.
  2. Right-click the player window and choose Open Location (or find it in the File menu).
  3. Paste the stream URL and click Open.
  4. Playback should start automatically (seeking may work depending on the version).

Other players

Most modern players (e.g., PotPlayer, KMPlayer, SMPlayer) support opening URLs. Look for options like Open URL or Open Network Stream.

Tip: You can also open the copied URL directly in your browser – the HTML5 video player will attempt to play it. However, seeking and controls might be limited compared to dedicated players.

Quality Control

By default, Vangrapf serves the best quality that is compatible with browsers (MP4/H.264 with audio). You can add the &quality= parameter to the stream URL to change it:

  • &quality=720 – 720p (if available)
  • &quality=best – highest quality (may lack audio if FFmpeg is not installed on the proxy server)
  • &quality=max – full quality with audio merged (requires FFmpeg on the server)

Example: https://proxy/stream?url=...&quality=720

Proxy Settings

To change the proxy server, click the ☰ icon in the top-right corner. You can:

  • Enter a custom proxy URL (e.g., your self-hosted Vangrapf backend).
  • Provide a custom URL for the proxy list (default is Vangrapf’s public list).
  • Leave the fields empty to use the default automatic proxy selection.

Settings are saved in your browser’s local storage.

Troubleshooting

  • Proxy not working? – Check if the proxy is reachable by visiting https://proxy/health. If not, choose another proxy.
  • Download interrupted? – Click Cancel and try again. Ensure you have enough disk space and a stable connection.
  • No results found? – Make sure you typed the query correctly and that the proxy is online. Some proxies might block search.
  • Video doesn't play in the player? – Verify that the copied URL includes the full https:// and that your player supports the media type (MP4).

Summary

Vangrapf gives you a simple way to watch and download YouTube videos without ads or regional restrictions. Use the copy‑link feature to play videos in your favourite player, or download them directly for offline viewing. The tool works with your own proxy or with the public proxy list provided by the Vangrapf project.

For more information, check the GitHub repository.
